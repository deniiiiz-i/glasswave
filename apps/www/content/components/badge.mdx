## Usage

```tsx
import { Badge } from "glasswave";
```

```tsx
<Badge>Default</Badge>
<Badge variant="outline">Outline</Badge>
<Badge variant="glass">Glass</Badge>
<Badge variant="destructive">Destructive</Badge>
```

## API Reference

### Badge

| Prop        | Type                                                 | Default     |
| ----------- | ---------------------------------------------------- | ----------- |
| `variant`   | `"default" \| "outline" \| "glass" \| "destructive"` | `"default"` |
| `className` | `string`                                             | —           |

Accepts all standard `<div>` attributes.
