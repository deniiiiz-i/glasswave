## Usage

```tsx
import { Alert, AlertTitle, AlertDescription } from 'glasswave';
```

```tsx
<Alert variant="default | success | warning | destructive">
  <AlertTitle>Heads up</AlertTitle>
  <AlertDescription>
    This is an alert with a title and description.
  </AlertDescription>
</Alert>
```

Each variant ships a matching system-color icon. Pass your own with the
`icon` prop, or `icon={null}` to hide it:

```tsx
<Alert icon={<Sparkles />}>
  <AlertTitle>Custom icon</AlertTitle>
  <AlertDescription>Any icon works here.</AlertDescription>
</Alert>
```

## API Reference

### Alert

| Prop        | Type                                                   | Default     |
| ----------- | ------------------------------------------------------ | ----------- |
| `variant`   | `"default" \| "destructive" \| "success" \| "warning"` | `"default"` |
| `icon`      | `ReactNode`                                            | —           |
| `className` | `string`                                               | —           |

Pass `icon={null}` to hide the leading icon. `AlertTitle` and `AlertDescription` accept `children` and `className`.
