## Usage

```tsx
import {
  InputOTP,
  InputOTPGroup,
  InputOTPSlot,
  InputOTPSeparator,
} from "glasswave";

<InputOTP maxLength={6}>
  <InputOTPGroup>
    <InputOTPSlot index={0} />
    …
  </InputOTPGroup>
  <InputOTPSeparator />
  <InputOTPGroup>…</InputOTPGroup>
</InputOTP>
```

## Props

`InputOTP` forwards to the underlying `OTPInput`: `maxLength`, `value`, `onChange`, `onComplete`, `pattern`, `containerClassName`, etc.

`InputOTPSlot` requires `index` (0-based) matching the slot position.

Package also exports `REGEXP_ONLY_DIGITS` and related patterns from `input-otp` if you import them from `input-otp` directly in app code.
