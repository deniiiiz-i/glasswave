## Usage

```tsx
import { Kbd } from "glasswave";

<Kbd>⌘</Kbd>
```

Renders a native `kbd` with subtle glass border and mono sizing. Accepts all standard `kbd` HTML attributes.

## API Reference

`Kbd` renders a `<kbd>` element and accepts all standard `<kbd>` attributes plus `className`.
