## Usage

```tsx
import {
  Table,
  TableHeader,
  TableBody,
  TableFooter,
  TableRow,
  TableHead,
  TableCell,
  TableCaption,
} from "glasswave";
```

`Table` wraps `<table>` in a rounded, bordered scroll container. Row hover uses `hover:bg-white/[0.04]`; `data-state="selected"` is supported on `TableRow`.

## Accessibility

Use `TableCaption` for summaries. Keep header cells as `TableHead` (`th`) with appropriate scope in complex tables.
