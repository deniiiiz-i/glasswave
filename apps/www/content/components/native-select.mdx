## Usage

```tsx
import { NativeSelect } from "glasswave";

<NativeSelect defaultValue="a">
  <option value="a">A</option>
  <option value="b">B</option>
</NativeSelect>
```

Extends native `select` props (`multiple`, `required`, `disabled`, `onChange`, etc.). Chevron is an embedded SVG (light/dark).

## API Reference

`NativeSelect` renders a styled native `<select>` and accepts all standard `<select>` attributes plus `className`.
