## Usage

```tsx
import {
  InputGroup,
  InputGroupAddon,
  InputGroupInput,
  InputGroupText,
} from "glasswave";

<InputGroup>
  <InputGroupAddon align="inline-start">
    <InputGroupText>https://</InputGroupText>
  </InputGroupAddon>
  <InputGroupInput placeholder="example.com" />
</InputGroup>
```

## API Reference

### InputGroupAddon

| Prop    | Type                             | Default          |
| ------- | -------------------------------- | ---------------- |
| `align` | `"inline-start" \| "inline-end"` | `"inline-start"` |

The other parts render their native element and accept its attributes plus `className`.
