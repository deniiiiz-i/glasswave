## Usage

```tsx
import {
  InputGroup,
  InputGroupAddon,
  InputGroupInput,
  InputGroupText,
} from "glasswave";

<InputGroup>
  <InputGroupAddon align="inline-start">
    <InputGroupText>https://</InputGroupText>
  </InputGroupAddon>
  <InputGroupInput placeholder="example.com" />
</InputGroup>
```

## Parts

| Part | Notes |
|------|--------|
| `InputGroup` | Glass field surface; flex row that holds the input and addons |
| `InputGroupAddon` | `align`: `"inline-start"` \| `"inline-end"`; sits inline next to the input |
| `InputGroupInput` | Transparent borderless input that fills the group |
| `InputGroupText` | Muted label inside addon |
