## Usage

```tsx
import { DirectionProvider, useDirection } from "glasswave";

<DirectionProvider dir="rtl">
  {children}
</DirectionProvider>
```

`useDirection(localDir?)` returns the effective `dir` (`"ltr"` \| `"rtl"`). Pair with `dir` on HTML elements for plain text/layout when Radix primitives are not involved.

## Props

`DirectionProvider`: `dir` (required), `children`.
