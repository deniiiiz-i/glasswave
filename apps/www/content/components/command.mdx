## Usage

```tsx
import {
  Command,
  CommandInput,
  CommandList,
  CommandEmpty,
  CommandGroup,
  CommandItem,
  CommandSeparator,
  CommandShortcut,
} from "glasswave";
```

`Command` accepts cmdk props: `label`, `shouldFilter`, `filter`, `value`, `onValueChange`, `loop`, `vimBindings`, etc.
