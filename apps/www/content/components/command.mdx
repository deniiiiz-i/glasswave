## Usage

```tsx
import {
  Command,
  CommandInput,
  CommandList,
  CommandEmpty,
  CommandGroup,
  CommandItem,
  CommandSeparator,
  CommandShortcut,
} from "glasswave";
```

`Command` accepts cmdk props: `label`, `shouldFilter`, `filter`, `value`, `onValueChange`, `loop`, `vimBindings`, etc.

## Parts

| Part | Purpose |
|------|---------|
| `CommandInput` | Search field (leading search icon) |
| `CommandList` | Scrollable results (`max-h` capped) |
| `CommandEmpty` | Shown when nothing matches |
| `CommandGroup` | Optional `heading` |
| `CommandItem` | Row; `onSelect`, `value`, `keywords` |
| `CommandSeparator` | Divider |
| `CommandShortcut` | Right-aligned hint (e.g. ⌘K) |

Use inside `PopoverContent` with `p-0` for combobox-style UIs.
