## Usage

```tsx
import {
  FieldGroup,
  Field,
  FieldLabel,
  FieldDescription,
  FieldError,
  Input,
} from "glasswave";
```

`FieldLabel` is the Radix `Label` primitive. `FieldError` sets `role="alert"` for screen readers.

## API Reference

`FieldGroup`, `Field`, `FieldDescription` and `FieldError` render native elements; `FieldLabel` wraps [Radix UI Label](https://www.radix-ui.com/primitives/docs/components/label). Each accepts its native attributes plus `className`.
