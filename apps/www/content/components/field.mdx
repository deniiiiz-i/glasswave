## Usage

```tsx
import {
  FieldGroup,
  Field,
  FieldLabel,
  FieldDescription,
  FieldError,
  Input,
} from "glasswave";
```

`FieldLabel` is the Radix `Label` primitive. `FieldError` sets `role="alert"` for screen readers.

## Layout

`FieldGroup` adds vertical spacing between multiple fields. Each `Field` is a column with gap between label, control, and helper text.
