## API Reference

### Button

| Prop        | Type                                                 | Default     |
| ----------- | ---------------------------------------------------- | ----------- |
| `variant`   | `"default" \| "outline" \| "glass" \| "destructive"` | `"default"` |
| `size`      | `"sm" \| "md" \| "lg" \| "icon"`                     | `"md"`      |
| `asChild`   | `boolean`                                            | `false`     |
| `className` | `string`                                             | —           |

Accepts all standard `<button>` attributes.
