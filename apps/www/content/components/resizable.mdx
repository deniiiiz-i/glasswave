## Usage

```tsx
import {
  ResizablePanelGroup,
  ResizablePanel,
  ResizableHandle,
} from "glasswave";

<ResizablePanelGroup orientation="horizontal">
  <ResizablePanel defaultSize={50}>A</ResizablePanel>
  <ResizableHandle withHandle />
  <ResizablePanel defaultSize={50}>B</ResizablePanel>
</ResizablePanelGroup>
```

## API

| Export | Maps to library |
|--------|-----------------|
| `ResizablePanelGroup` | `Group` |
| `ResizablePanel` | `Panel` |
| `ResizableHandle` | `Separator` |

`ResizableHandle` supports `withHandle` to show a grip. Pass `orientation` to the group (`horizontal` \| `vertical`). See [react-resizable-panels](https://github.com/bvaughn/react-resizable-panels) for `defaultLayout`, `onLayoutChanged`, `collapsible`, etc.
