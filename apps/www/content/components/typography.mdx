## Usage

```tsx
import {
  TypographyH1,
  TypographyH2,
  TypographyH3,
  TypographyH4,
  TypographyP,
  TypographyLead,
  TypographyLarge,
  TypographySmall,
  TypographyMuted,
  TypographyBlockquote,
  TypographyList,
  TypographyInlineCode,
} from "glasswave";
```

Each export is a thin styled wrapper around the matching HTML element (`h1`–`h4`, `p`, `small`, `ul`, `code`, etc.). Pass `className` to extend.

`TypographyList` renders a `ul` with disc markers and vertical rhythm.

## API Reference

Each component (`TypographyH1`–`TypographyH4`, `TypographyP`, `TypographyLead`, `TypographyLarge`, `TypographySmall`, `TypographyMuted`, `TypographyBlockquote`, `TypographyList`, `TypographyInlineCode`) renders its matching semantic element and accepts its native attributes plus `className`.
