## Usage

```tsx
import { ToggleGroup, ToggleGroupItem } from "glasswave";
```

```tsx
<ToggleGroup type="multiple">
  <ToggleGroupItem value="bold" aria-label="Toggle bold">
    <FontBoldIcon className="h-4 w-4" />
  </ToggleGroupItem>
  <ToggleGroupItem value="italic" aria-label="Toggle italic">
    <FontItalicIcon className="h-4 w-4" />
  </ToggleGroupItem>
  <ToggleGroupItem value="strikethrough" aria-label="Toggle strikethrough">
    <FontStrikethroughIcon className="h-4 w-4" />
  </ToggleGroupItem>
</ToggleGroup>
```

## API Reference

Built on [Radix UI Toggle Group](https://www.radix-ui.com/primitives/docs/components/toggle-group). `variant` and `size` set on `ToggleGroup` cascade to its items.

### ToggleGroup

| Prop      | Type                        | Default     |
| --------- | --------------------------- | ----------- |
| `variant` | `"default" \| "secondary"`  | `"default"` |
| `size`    | `"default" \| "sm" \| "lg"` | `"default"` |
