## API Reference

### Toast

| Prop       | Type                                                                                              | Default       |
| ---------- | ------------------------------------------------------------------------------------------------- | ------------- |
| `message`  | `string`                                                                                          | —             |
| `type`     | `"default" \| "error"`                                                                            | `"default"`   |
| `duration` | `number`                                                                                          | `5000`        |
| `position` | `"top-left" \| "top-center" \| "top-right" \| "bottom-left" \| "bottom-center" \| "bottom-right"` | `"top-right"` |
