## Usage

```tsx
import { Combobox, type ComboboxOption } from "glasswave";

const options: ComboboxOption[] = [
  { value: "a", label: "Alpha", keywords: ["first"] },
  { value: "b", label: "Beta" },
];

<Combobox options={options} value={value} onValueChange={setValue} />
```

## API Reference

### Combobox

| Prop                | Type                              | Default     |
| ------------------- | --------------------------------- | ----------- |
| `options`           | `ComboboxOption[]`                | —           |
| `value`             | `string \| null`                  | —           |
| `onValueChange`     | `(value: string \| null) => void` | —           |
| `placeholder`       | `string`                          | `"Select…"` |
| `searchPlaceholder` | `string`                          | —           |
| `emptyText`         | `string`                          | —           |
| `disabled`          | `boolean`                         | `false`     |

`ComboboxOption` is `{ value: string; label: string; keywords?: string[] }`.
