## Usage

```tsx
import { Combobox, type ComboboxOption } from "glasswave";

const options: ComboboxOption[] = [
  { value: "a", label: "Alpha", keywords: ["first"] },
  { value: "b", label: "Beta" },
];

<Combobox options={options} value={value} onValueChange={setValue} />
```

## Props (`Combobox`)

| Prop | Type | Description |
|------|------|-------------|
| `options` | `ComboboxOption[]` | `value`, `label`, optional `keywords` for cmdk filter |
| `value` | `string \| null` | Selected value |
| `onValueChange` | `(v: string \| null) => void` | Change handler |
| `placeholder` | `string` | Trigger label when empty |
| `searchPlaceholder` | `string` | Command input hint |
| `emptyText` | `string` | Empty state copy |
| `disabled` | `boolean` | Disables trigger |

For full control, compose `Popover`, `Button`, and `Command` manually (see Command docs).
