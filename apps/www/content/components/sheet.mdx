## Usage

```tsx
import {
  Sheet,
  SheetTrigger,
  SheetContent,
  SheetHeader,
  SheetTitle,
  SheetDescription,
} from "glasswave";

<Sheet>
  <SheetTrigger asChild>
    <Button>Open</Button>
  </SheetTrigger>
  <SheetContent side="right">…</SheetContent>
</Sheet>
```

## `SheetContent`

| Prop | Type | Default |
|------|------|---------|
| `side` | `"top"` \| `"right"` \| `"bottom"` \| `"left"` | `"right"` |

Includes overlay, close button, and glass surface. `SheetPortal`, `SheetOverlay`, `SheetClose`, `SheetFooter` are exported for custom composition.
