## Usage

```tsx
import {
  Item,
  ItemMedia,
  ItemContent,
  ItemTitle,
  ItemDescription,
  ItemActions,
} from "glasswave";
```

## Props

| Prop | Element | Notes |
|------|---------|--------|
| `asChild` | `Item` | Merge styles onto child via `Slot` |
| `variant` | `Item` | `"default"` \| `"glass"` |
| `variant` | `ItemMedia` | `"icon"` \| `"image"` |

`ItemActions` is a shrink-0 container for trailing buttons.
