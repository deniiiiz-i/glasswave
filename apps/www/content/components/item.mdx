## API Reference

### Item

| Prop      | Type                   | Default     |
| --------- | ---------------------- | ----------- |
| `variant` | `"default" \| "glass"` | `"default"` |
| `asChild` | `boolean`              | `false`     |

### ItemMedia

| Prop      | Type                | Default  |
| --------- | ------------------- | -------- |
