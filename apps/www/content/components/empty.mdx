## Usage

```tsx
import {
  Empty,
  EmptyHeader,
  EmptyMedia,
  EmptyTitle,
  EmptyDescription,
  EmptyContent,
} from "glasswave";
```

## Parts

| Part | Role |
|------|------|
| `Empty` | Dashed bordered container |
| `EmptyHeader` | Stacks title + description |
| `EmptyMedia` | `variant`: `"default"` (boxed icon) or `"icon"` (plain) |
| `EmptyTitle` | Heading |
| `EmptyDescription` | Supporting text |
| `EmptyContent` | CTAs row |
